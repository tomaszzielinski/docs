Speeding up you Django website by tomaszzielinski

Rarely-known (and/or undocumented) Django features

  • form.Form.has_changed() - checks if the form data is different that the initial data
  • django.utils.html.linebreaks(...) - converts newlines into <p> and <br> tags
  • django.utils.html.urlize(...) - safely converts URLs into clickable links. This is a hard task otherwise: 1, 2, 3, 4.
  • model.Meta.order_with_respect_to
  • model.Meta.order_with_respect_to
  • The difference between Model.objects.filter(a__x=1, a__y=2) and Model.objects.filter(a__x=1).filter(a__y=2)
